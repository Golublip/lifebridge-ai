# Security Policy

## Supported Versions
We support security patches for the following versions:
- `v1.x` (Active Development)

## Reporting a Vulnerability
We take the security of LifeBridge AI seriously. If you find a security vulnerability, please do NOT open a public GitHub issue. Instead, report it by emailing the maintenance team.

All reports will be acknowledged within 48 hours, and a patch release will be coordinated.
