# Contributing to LifeBridge AI

First off, thank you for considering contributing to LifeBridge AI! It's people like you who make LifeBridge AI such a powerful tool for global health equity.

## Code of Conduct
By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).

## How Can I Contribute?
1. **Reporting Bugs**: Open an issue using the Bug Report template.
2. **Suggesting Enhancements**: Open an issue using the Feature Request template.
3. **Pull Requests**:
   - Fork the repository.
   - Create a branch for your fix/feature: `git checkout -b feature/amazing-feature`.
   - Make sure unit tests pass: `pytest backend/tests/`.
   - Commit your changes: `git commit -m 'Add amazing feature'`.
   - Push to the branch: `git push origin feature/amazing-feature`.
   - Open a Pull Request.

## Coding Style
- Write clean, modular Python and React code.
- Follow PEP 8 guidelines for Python.
- Maintain safety guardrails: Never allow agents to make diagnoses.
