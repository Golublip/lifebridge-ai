# Frequently Asked Questions (FAQ)

---

### Q: Does LifeBridge AI provide diagnoses?
**A:** No. Safety is our primary design constraint. LifeBridge AI strictly provides educational information, clinic lookup, compliance helpers, and emergency bypass actions. Under no circumstances does it replace medical practitioners.

---

### Q: How does the system handle missing API keys?
**A:** If `GEMINI_API_KEY` is not present, the `GeminiClient` automatically defaults to heuristic simulations/mocks to ensure that frontend features and demo scripts run seamlessly for grading and development.

---

### Q: What languages are supported?
**A:** LifeBridge AI currently supports English, Spanish, Swahili, and Hindi. Set the user preference in the settings panel to change the UI and chat localization.
